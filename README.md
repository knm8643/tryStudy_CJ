## ✅ commit 규칙
- commit 메세지: [문제 출처(플랫폼)] 문제이름 / 난이도 / 걸린시간
- 터미널에서 작성법: 
```
git commit -m "[BOJ] Hello World / 브론즈5 / 1분"
```
- 플랫폼 작성법 통일: 
  * [BOJ] - 백준 
  * [PGS] - 프로그래머스
  * [LTC] - 리트코드
  * [CFS] - 코드포스
  * [SEA] - 삼성SW Expert Academy
